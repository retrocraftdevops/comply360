## 2. Name Reservation

### 2.1 Check Name Availability
1. Go to **Registrations** > **New** > **Name Reservation**
2. Enter proposed name
3. System checks availability in real-time
4. Review similar names and suggestions

### 2.2 Reserve Company Name
1. Select jurisdiction (SA or Zimbabwe)
2. Enter applicant details
3. Upload required documents (ID, proof of address)
4. Review and submit
5. Track status in dashboard

**Processing Time:** 2-5 business days  
**Valid For:** 60 days (SA), 30 days (Zimbabwe)

## 3. Company Registration

### 3.1 Pty Ltd Registration (Step-by-Step)

**Step 1: Company Details**
- Select reserved name
- Enter business address
- Choose SIC code
- Set financial year end

**Step 2: Directors & Shareholders**
- Add director information
- Upload ID documents
- Allocate shares
- AI auto-fills from ID (name, DOB)

**Step 3: Share Capital**
- Set authorized capital
- Define share classes
- Allocate shareholding

**Step 4: MOI (Memorandum)**
- Choose standard or custom MOI
- Configure key provisions
- Preview before proceeding

**Step 5: Documents**
- Upload all required documents
- System verifies automatically
- AI checks for completeness

**Step 6: Payment**
- Review pricing breakdown
- Select payment method
- Complete payment securely

**Step 7: Submit & Track**
- Submit to CIPC/DCIP
- Receive confirmation email
- Track status in real-time

## 9. Troubleshooting

### Common Issues

**Cannot Login:**
- Check email/password (case-sensitive)
- Clear browser cache
- Try password reset
- Contact support if persists

**Document Upload Fails:**
- Check file size (max 10MB)
- Ensure supported format (PDF, JPG, PNG)
- Check internet connection
- Try different browser

**Payment Not Processing:**
- Verify card details
- Check sufficient funds
- Try alternative payment method
- Contact support with error code

## 10. FAQs

**Q: How long does company registration take?**  
A: Standard: 5-10 business days. Expedited: 24-48 hours.

**Q: Can I edit a submitted registration?**  
A: No. Contact support to withdraw and resubmit.

**Q: How do I change my company name?**  
A: Start new name reservation, then submit amendment.

**Q: What documents are required?**  
A: ID copies (all directors), proof of address, MOI, consent forms.

**Q: How secure is my data?**  
A: Bank-level encryption (AES-256), SOC 2 certified.
